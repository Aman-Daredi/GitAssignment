Command to initialize git repository : git init

Command to clone remote repository : git clone <remote_repository_link> .

Command to stage all changes : git add .

Command to view state of working directory and staging area : git status

Command to create new branch : git branch <new_branch_name>

Command to change branch : git checkout <branch_name>

Command to clean staging area: git reset --staged .

Command to get commits and changes from remote branch (It should not merge changes in current branch) : git fetch <remote_name> <branch_name>

Command to get commits and changes from remote branch (It should merge changes in current branch) : git merge <remote_name>/<branch_name>

Command to cherry pick commit which SHA => 092018283103810930 : git cherry-pick 092018283103810930

Command to push branch : git push <remote> <branch>

Command to delete remote branch: git push <remote_name> -d <branch_name>

Command to do iterative rebase: git rebase -i HEAD~<variable_num>
